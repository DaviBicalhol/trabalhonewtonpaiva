# Tec-Newton - E-commerce de Tecnologia

Este projeto é um site de compras de eletrônicos desenvolvido para a disciplina de Programação WEB. O foco foi criar uma interface moderna, responsiva e totalmente navegável utilizando apenas tecnologias front-end.

## Objetivo
O sistema simula uma jornada de compra completa: desde a visualização de produtos em destaque na Home, passando pela listagem geral, simulação de carrinho de compras e um canal de comunicação via formulário de contato.

## Tecnologias e Recursos
- **HTML5**: Estruturação semântica (nav, main, section, article, footer, form).
- **CSS3**: 
  - **Flexbox**: Para layouts flexíveis e alinhamento de cards.
  - **Animação de Gradiente**: Fundo dinâmico para uma estética Tech.
  - **Responsividade**: Uso de @media queries para adaptação em celulares.
- **GitHub**: Controle de versão e hospedagem.

## Estrutura de Páginas
- `index.html`: Home com destaques e últimas notícias.
- `produtos.html`: Catálogo com todos os itens disponíveis.
- `carrinho.html`: Tela de simulação de itens selecionados (Tabela HTML).
- `contato.html`: Página com formulário de atendimento ao cliente.
- `style.css`: Estilo centralizado para todo o projeto.

## Como Executar o Projeto
1. Faça o download do repositório ou realize um `git clone`.
2. Abra a pasta do projeto no seu computador.
3. Clique duas vezes no arquivo `index.html` para abrir no seu navegador de preferência.
4. (Opcional) Acesse o link do **GitHub Pages** para visualizar a versão online.
